RFID Basics with RC522

Learn how to wire the RC522 RFID module, read tag UIDs, and handle multiple tags with Arduino.

What is RFID?

RFID (Radio Frequency Identification) is a wireless communication technology used to identify and track objects using radio waves. It consists of a reader and RFID tags. The RC522 module is one of the most commonly used RFID readers in DIY and IoT projects.

How RFID Works

The RFID reader (RC522) generates a magnetic field that powers the RFID tag when it comes close. The tag then sends its unique ID back to the reader. This data can be used for access control, attendance systems, inventory tracking, and more.

RC522 Pinout

The RC522 communicates using the SPI protocol. Here’s the pin mapping:

  • SDA → Arduino Pin 10
  • SCK → Arduino Pin 13
  • MOSI → Arduino Pin 11
  • MISO → Arduino Pin 12
  • IRQ → Not connected
  • GND → GND
  • RST → Arduino Pin 9
  • 3.3V → 3.3V Power

Arduino Code for RFID Reading

Upload the following code to your Arduino board to read RFID tag UIDs:


#include <SPI.h>
#include <MFRC522.h>

#define SS_PIN 10
#define RST_PIN 9
MFRC522 rfid(SS_PIN, RST_PIN);

void setup() {
  Serial.begin(9600);
  SPI.begin();
  rfid.PCD_Init();
  Serial.println("Scan an RFID tag...");
}

void loop() {
  if (!rfid.PICC_IsNewCardPresent() || !rfid.PICC_ReadCardSerial()) return;
  
  Serial.print("UID Tag: ");
  for (byte i = 0; i < rfid.uid.size; i++) {
    Serial.print(rfid.uid.uidByte[i], HEX);
    Serial.print(" ");
  }
  Serial.println();
  delay(1000);
}
